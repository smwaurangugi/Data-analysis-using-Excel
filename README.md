# Sales Operations analysis using excel.
* The data shared had missing values and below are the approaches/ assumptions used to fill the missing values.
* Missing City. The approach used is matching previous orders to the given row as most of the customers are repeat customers, I used details from previous orders.
* Missing sales person. The approach used is matching previous orders to the given row as most of the customers are repeat customers, I used details from previous orders.
* Missing Channel. The approach used is matching previous orders to the given row as most of the customers are repeat customers, I used details from previous orders.
* There was evidence of incorrect unit prices. The approach I applied here is getting the mean of prices from previous orders.
* The maximum discount was capped at 30%.
